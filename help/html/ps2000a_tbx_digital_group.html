Digital Group

The Digital group contains a set of properties and functions related to the digital ports of the device (applicable to MSO models only).

Obtaining a Digital Group Object from the Instrument Driver

Below is an example of how to obtain a Digital group object from the Instrument Driver device object. This step only has to be carried out once prior to accessing properties or invoking group object functions:

    digitalGroupObj = get(ps2000aDeviceObj, 'Digital');
    digitalGroupObj = digitalGroupObj(1);

Properties

The following properties are defined for the Device object:

Property Name Data type Default value Can be changed by User Description
digitalPortSettings Double 0 No A matrix containing the port settings for digital ports. (Do not use)
maxLogicLevel Double 32767 No The maximum count level for a digital port (corresponding to +5 V)
minLogicLevel Double -32767 No The minimum count level for a digital port (corresponding to -5 V)

Use the set and get functions, where appropriate, to change the value of a property or retrieve it's value.

Example:

    % Obtain the maximum logic level in ADC counts
    maxLogicLevel = get(digitalGroupObj, 'maxLogicLevel');
