Device Object

The Instrument Driver device object contains a set of properties and functions in order to configure the device (regardless of which data acquisition mode or whether the signal generator is being used) and query certain parameters.

Properties

The following properties are defined for the Device object:

Property Name Data type Default value Can be changed by User Description
bandwidth Double 0 No Represents the maximum bandwidth (3dB cut-off point) of the oscilloscope in hertz
bufferMemory Double 0 No Defines the size of the memory buffer on the device (in samples)
channelCount Double 0 No The number of analog input channels on the device.
channelSettings Any 0 No A matrix containing the channel settings for analogue channels (Do not use)
digitalPortCount Double 0 No The number of digital ports on the device
displayOutput Double (Enumeration) 1 Yes Enumeration value to select whether to display output from function calls in the MATLAB Command Window
firstRange Double (Enumeration) 2 No Enumeration corresponding to the lowest voltage range supported by the device (refer to the enPS2000ARange enumeration values)
hasHardwareEts Double (Enumeration) 0 No Indicates if the device has hardware-based Equivalent Time Sampling
lastRange Double (Enumeration) 10 No Enumeration corresponding to the highest voltage range supported by the device (refer to the enPS2000ARange enumeration values)
maxADCValue Double 32512 No The maximum ADC count value for the device
maxSamplingRate Double 0 No The maximum single shot sampling rate for the device (samples per second)
minADCValue Double -32512 No The minimum ADC count value for the device
numPostTriggerSamples Double 0 Yes The number of samples to store after a trigger event
numPreTriggerSamples Double 0 Yes The number of samples to store after a trigger event
timebase Double 65 Yes The timebase used for the scope when capturing data in Block and Rapid Block mode (see also Timebases)
unitHandle Double 0 No The handle assigned by the underlying ps2000a/libps2000a shared library for the device
unitSerial String None No The device batch/serial number

Use the set and get functions, where appropriate, to change the value of a property or retrieve it's value.

Examples:

    % Obtain the number of analogue channels on the device
    numChannels = get(ps2000aDeviceObj, 'channelCount');

    % Set the number of post-trigger samples to collect
    set(ps2000aDeviceObj, 'numPostTriggerSamples', 1000000);
