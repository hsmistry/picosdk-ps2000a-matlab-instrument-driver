Overview

PicoScope 2000 Series oscilloscopes supported by this toolbox also have a signal generator with the following features:

  • In-built waveforms e.g. sine, square and ramp-up
  • Arbitrary waveform generator allowing a User to specify a custom waveform to be output
  • Sweep the signal generator output frequency
  • Output a specified number of cycles or sweeps using a trigger

Using the signal generator

Below is the general procedure for using the signal generator:

  1. Create a device object for the Instrument using icdevice.
  2. Connect to the Instrument using connect.
  3. Configure the input channels and trigger conditions as required.
  4. Set the offsetVoltage, peakToPeakVoltage, startFrequency and stopFrequency properties of the Signalgenerator group as required.
  5. Use setSigGenArbitrary or sigGenArbitrarySimple to output an arbitrary waveform, or use setSigGenBuiltIn or sigGenBuiltInSimple to output a built-in waveform.
  6. Start any data collection as required.
  7. While waiting for a trigger, set the Signalgenerator group properties from step 4 if they need changing and call setSigGenPropertiesArbitrary or setSigGenPropertiesBuiltIn if the signal generator properties need to be changed.
  8. If setSigGenArbitrary or setSigGenBuiltIn have been configured to output a number of shots or sweeps when a software trigger occurs call ps2000aSigGenSoftwareControl.
  9. Use setSigGenOff to set the output to a 0 V DC signal.
  10. Disconnect from the Instrument using disconnect.
  11. Use the delete function to clean up.

AWG Index Modes

The arbitrary waveform generator supports single and dual index modes to help you make the best use of the waveform buffer.

Single mode. The generator outputs the raw contents of the buffer repeatedly. This mode is the only one that can generate asymmetrical waveforms. You can also use this mode for symmetrical waveforms, but the dual mode make more efficient use of the buffer memory.

Dual mode. The generator outputs the contents of the buffer from beginning to end, and then does a second pass in the reverse direction through the buffer. This allows you to specify only the first half of a waveform with twofold symmetry, such as a Gaussian function, and let the generator fill in the other half.
