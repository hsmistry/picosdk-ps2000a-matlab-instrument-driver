Streaming Group

The Streaming group contains a set of properties and functions related to the streaming mode capture functionality of the device.

Obtaining a Streaming Group Object from the Instrument Driver

Below is an example of how to obtain a Streaming group object from the Instrument Driver device object. This step only has to be carried out once prior to accessing properties or invoking group object functions:

    streamingGroupObj = get(ps2000aDeviceObj, 'Streaming');
    streamingGroupObj = streamingGroupObj(1);

Properties

The following properties are defined for the Device object:

Property Name Data type Default value Can be changed by User Description
autoStop Double (Enumeration) 1 Yes A flag to specify if streaming data collection should stop when the sum of pre-trigger and post-trigger samples have been taken. This flag should be set to 0 or 1.
streamingInterval Double 0 Yes The streaming mode sample interval to be used (in seconds).

Use the set and get functions, where appropriate, to change the value of a property or retrieve it's value.

Examples:

    % Obtain the current streaming interval setting (in seconds)
    streamingInterval = get(streamingGroupObj, 'streamingInterval');

    % Set the autoStop property to stop when the pre- and post-trigger
    % samples have been collected.
    set(streamingGroupObj, 'autoStop', 1);
