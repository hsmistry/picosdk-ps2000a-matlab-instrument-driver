Connecting and Disconnecting

This page describes how to connect to and disconnect from a PicoScope 2000 Series (A API) Oscilloscope or Mixed Signal Oscilloscope using the the PicoScope 2000 Series (A API) - MATLAB Generic Instrument Driver.

Contents

  • Connecting to an Instrument
  • Disconnecting from an Instrument
  • Working with Multiple Instruments

Connecting to an Instrument

Prior to attempting to establish a connection with the instrument, run the PS2000aConfig.m script.

PS2000aConfig;

% Create a device object using the
% <matlab:doc('/instrument/icdevice') |icdevice|> function.
% The Instrument Driver name should be passed as the argument and a second
% optional argument can be used to specify the serial number of the device
% as shown below:

    ps2000aDeviceObj = icdevice('picotech_ps2000a_generic.mdd', 'TT003/0010');

Disconnecting from an Instrument

On completion of the tasks for the instrument session, disconnect from the instrument using the disconnect function and use the delete function to clean up.

    disconnect(ps2000aDeviceObj);
    delete(ps2000aDeviceObj);

Connection to PicoScope 2408B with serial number TT003/0010 closed successfully.
Libraries unloaded successfully.

Working with Multiple Instruments

The picotech_ps2000a_generic.mdd Instrument Driver can support multiple instruments but this is only recommended for Block and Rapid block mode captures as well as use of the signal generator.

Due to a limitation with the ps2000aWrap/libps2000aWrap shared libraries, it is not recommended to attempt to collect data in Streaming mode with multiple Instruments.

To use multiple instruments in a session, use the icdevice function to create a device object for each instrument (specifying the serial numbers as required), then call the connect, disconnect and delete functions as described above.

Due to the polling mechanism used to check if the Instrument has collected data in Block and Rapid block modes, the use of Parallel Computing Toolbox functions may be worth considering when using multiple Instruments.

    % Create device objects.
    ps2000aDeviceObj = icdevice('picotech_ps2000a_generic.mdd');
    ps2000aDeviceObj2 = icdevice('picotech_ps2000a_generic.mdd');

    % Connect device objects to hardware.
    connect(ps2000aDeviceObj);
    connect(ps2000aDeviceObj2);

    % Perform some tasks...

    % Disconnect and clean up.
    disconnect(ps2000aDeviceObj);
    delete(ps2000aDeviceObj);

    disconnect(ps2000aDeviceObj2);
    delete(ps2000aDeviceObj2);
