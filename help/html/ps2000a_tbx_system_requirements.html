PC Requirements

To ensure that your PicoScope 2000 Series PC Oscilloscope operates correctly with the MATLAB Instrument Driver, you must have a computer with at least the minimum system requirements to run one of the supported operating systems, as shown in the following table. The performance of the oscilloscope will be better with a more powerful PC, and will benefit from a multi-core processor.

USB 1.1 port (absolute minimum)

  • PicoScope oscilloscopes will operate slowly on a USB 1.1 port. Not recommended. USB 3.0 connections will run at about the same speed as USB 2.0.

USB

The ps2000a/libps2000a shared library offers four different sampling modes (methods of recording data) all of which support USB 1.1, USB 2.0 and USB 3.0. The fastest transfer rates are achieved using USB 2.0 or USB 3.0.

Note: USB 3.0 connections will run at about the same speed as USB 2.0.
