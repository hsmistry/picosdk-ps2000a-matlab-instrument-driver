Default Settings

On connection, the Instrument Driver is initialized with default settings as detailed on this page.

Input Channels

Analog channels

All analog channels are initialized as below:

  • Enabled, DC coupled, ±5 V range, and 0 V analog offset

Use the ps2000aSetChannel function to turn off any analog channels.

Digital ports (MSOs only):

All digital ports are initialized as below:

  • Enabled, + 1.5 V threshold

Use the ps2000aSetDigitalPort function in the Digital group to turn off any digital ports.

Triggering

  • Triggering is disabled by default.

Data Collection Parameters

The default data collection parameters are shown below:

Block/Rapid block

  • Timebase index : 64 (see Timebases)
  • Number of pre-trigger samples: 0
  • Number of post-trigger samples: 8192

These values correspond to properties of the Block group.

Note: No segmentation is appplied to the device buffer memory. Use the ps2000aMemorySegments function to do this.

Equivalent Time Sampling (ETS) Mode

  • ETS mode data collection is switched off.

Streaming

  • Sampling interval: 1 microsecond
  • Auto stop: 1 (TRUE)

These values correspond to properties of the Streaming group.

Signal Generator Parameters

The following are default values of the signal generator:

  • Start frequency: 1000 Hz
  • Stop frequency: 1000 Hz
  • Offset voltage: 0 mV
  • Peak to Peak voltage: 2000 mV

These values correspond to properties of the Signalgenerator group.
