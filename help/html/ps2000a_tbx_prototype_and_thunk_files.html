Prototype and Thunk Files

The PicoScope 2000 Series (A API) - MATLAB Generic Instrument Driver Toolbox folder structure contains folders specific to platforms containing prototype files and thunk files.

Prototype files contain information extracted from the ps2000aApi.h and ps2000aWrap.h C header files provided as part of the Pico Technology Software Development Kit which includes function, enumeration and structure definitions. Each supported platform has its own set of prototypes files corresponding to these header files.

Thunk files provide a compatibility layer to shared library files for 64-bit versions of MATLAB.

The table below summarises the prototype and thunk files provided as part of the toolbox for MATLAB versions for Microsoft Windows and Linux platforms.

Platform Architecture Prototype files Thunk files
Microsoft Windows win32 ps2000aMFile_win32.m
ps2000aWrapMFile_win32.m		 N/A
win64 ps2000aMFile_win64.m
ps2000aWrapMFile_win64.m		 ps2000a_thunk_pcwin64.dll
ps2000aWrap_thunk_pcwin64.dll
Linux glnxa64 ps2000aMFile_glnxa64.m
ps2000aWrapMFile_glnxa64.m		 ps2000a_thunk_glnxa64.so
ps2000aWrap_thunk_glnxa64.so

Prototype and thunk files should be included when packaging code involving the PicoScope 2000 Series (A API) - MATLAB Generic Instrument Driver (please refer to Deploying Standalone Applications).
