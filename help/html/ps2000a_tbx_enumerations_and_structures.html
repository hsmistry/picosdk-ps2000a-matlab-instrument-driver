Enumerations and Structures

The ps2000aApi.h C header file installed as part of the PicoSDK defines enumeration values and structures that may be used when calling the API driver functions. The ps2000aWrap.h also defines some enumeration values (and has a structure defined for use by the ps2000aWrap shared library).

The enumeration and structure definitions are accessible from the MATLAB prototype files ps2000aMFile and ps2000aWrapMFile and can be used when calling Instrument Driver functions.

The PS2000aConfig.m file shows how to access the structures containing the enumeration and structure information:

[ps2000aMethodinfo, ps2000aStructs, ps2000aEnuminfo, ps2000aThunkLibName] = ps2000aMFile;

[ps2000aWrapMethodinfo, ps2000aWrapStructs, ps2000aWrapEnuminfo, ps2000aWrapThunkLibName] = ps2000aWrapMFile;

Enumerations

To view a list of enumeration types, type ps2000aEnuminfo (if you change the name of this structure, use that instead) at the MATLAB command prompt and press Enter:

ps2000aEnuminfo

Use the dot notation to access the different enumeration types and their range of values e.g. for an enumeration representing a ±1 V range:

oneVoltEnumVal = ps2000aEnuminfo.enPS2000ARange.PS2000A_1V

This value can then be used with the ps2000aSetChannel function.

Structures

To view a list of enumeration types, type ps2000aStructs (if you change the name of this structure, use that instead) at the MATLAB command prompt and press Enter:

ps2000aStructs

To obtain an instance of structure, assign it to a variable:

triggerConditionStruct = ps2000aStructs.tPS2000ATriggerConditions

To view a list of the fields forming the structure, view the members sub-structure:

ps2000aStructs.tPS2000ATriggerConditions.members

As shown above, each of the members requires an enPS2000ATriggerState enumeration value. Below is an example of assigning such a value to a field in the structure:

triggerConditionStruct.members.channelA = ps2000aEnuminfo.enPS2000ATriggerState.PS2000A_CONDITION_TRUE;

% Verify the value set
triggerConditionStruct.members
