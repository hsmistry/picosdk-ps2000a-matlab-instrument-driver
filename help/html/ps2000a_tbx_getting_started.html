Requirements

To use the PicoScope 2000 Series (A API) - MATLAB Generic Instrument Driver Toolbox, you must install the following hardware and software components:

  • A PicoScope 2000 Series Oscilloscope or Mixed Signal Oscilloscope (see Supported Hardware Models below)
  • MATLAB R2016b or higher (R2015b recommended for 32-bit versions of MATLAB on Microsoft Windows platforms)
  • Instrument Control Toolbox (v3.10 or later; v3.8 with MATLAB 2015b)
  • Pico Technology Software Development Kit (SDK) for your platform
  • PicoScope Support Toolbox
  • A supported operating system (see System Requirements)

Supported Hardware Models

The following PicoScope 2000 Series Models are supported by the Instrument Driver in this toolbox:

  • PicoScope 2206/7/8 (including ‘A’ and 'B' models)
  • PicoScope 2405A
  • PicoScope 2406/7/8 'B' models
  • PicoScope 2205 MSO and 2205A MSO
  • PicoScope 2206/7/8B MSO models

Note: This Instrument Driver does not support the PicoScope 2104, 2105, 2204A and 2205A models. These will require the PicoScope 2000 Series - MATLAB Generic Instrument Driver.

Pico Technology Software Development Kit Information

Pico Technology provides drivers (shared libraries) for the following platforms:

  • Microsoft Windows
  • Linux
  • Mac OS X

Drivers are provided as 32-bit and 64-bit versions, although support for Linux is limited to 64-bit versions of MATLAB. A 64-bit version of MATLAB is recommended for Mac OS X platforms.

Run the PS2000aSetup.m script included in this toolbox and follow the prompts to download and install the SDK for your platform.

Alternatively, below are platform specific guidelines:

Microsoft Windows:

  • For 32-bit versions of MATLAB please download and install the 32-bit Pico Technology SDK.
  • For 64-bit versions of MATLAB please download and install the 64-bit Pico Technology SDK.
  • Both are available to download from the Pico Technology Downloads page.

Note: PicoSDK 10.6.11 will be the last support version for Windows XP (SP3) and Vista operating systems.

Linux:

  • Download the libps2000a and libpswrappers packages using the instructions availble on the Linux Software & Drivers for Oscilloscopes and Data Loggers web page.

Mac OS X:

To use the PicoScope 2000 Series (A API) - MATLAB Generic Instrument Driver Toolbox on Mac OS X platforms, you will need to install the following:

  • PicoScope 6 for Mac OS X
  • A supported compiler

Please contact our Technical Support Team for further assistance with Mac OS X platforms.

As part of the installation process on Microsoft Windows and Linux platforms, the locations of the shared libraries should be added to the system path.

Installing PicoScope Hardware

To connect the data logger, plug the cable provided into any available USB port on your PC.

Microsoft Windows platforms:

  • The first time you connect the unit, Windows may display a New Hardware Wizard.
  • Follow any instructions in the Wizard and wait for the driver to be installed. Later versions of Windows display an Installing new hardware message and complete the process automatically.
  • Verify that a PicoScope 2000 Series Oscilloscope device appears in the Windows Device Manager.

Linux:

  • Open a Terminal window, type lsusb at the command prompt and press ENTER.
  • A PicoScope 2000A series PC Oscilloscope entry should be in the list that is displayed.

Mac OS X:

  • Open the System Information or System Profiler app
  • Click on USB under the Hardware item in the left pane
  • Verify that the device is present in the USB Device Tree
