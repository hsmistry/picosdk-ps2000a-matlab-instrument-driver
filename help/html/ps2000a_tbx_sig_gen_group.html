Signalgenerator Group

The Signalgenerator group contains a set of properties and functions related to the signal generator functionality of the device.

Obtaining a Signalgenerator Group Object from the Instrument Driver

Below is an example of how to obtain a Signalgenerator group object from the Instrument Driver device object. This step only has to be carried out once prior to accessing properties or invoking group object functions:

    sigGenGroupObj = get(ps2000aDeviceObj, 'Signalgenerator');
    sigGenGroupObj = sigGenGroupObj(1);

Properties

The following properties are defined for the Signalgenerator object:

Property Name Data type Min. Value Max. Value Default value Can be changed by User Description
awgBufferSize Double (Bounded) 16384 0 32768 No The size of the arbitrary waveform buffer on the device, in samples.
dacFrequency Double (Bounded) 2.0E7 0 2.0E7 No The update frequency of the Arbitrary Waveform Generator (MS/s or MHz)
ddsFrequency Double 0 N/A N/A No ddsFrequency is the direct digital synthesis frequency for the arbitrary waveform generator (in Hz). This property is only set for the PicoScope 2205 MSO, and is set to 0 for other models.
offsetVoltage Double (Bounded) 0 -2000.0 2000.0 Yes The voltage offset, in millivolts, to be applied to the waveform. The amplitude of the waveform should not exceed +/-2000 mV.
peakToPeakVoltage Double (Bounded) 2000.0 0.0 4000.0 Yes The peak-to-peak voltage, in millivolts, of the waveform signal. The amplitude of the waveform should not exceed +/-2000 mV.
sigGenType Double (Enum) 0 2 0 No The type of Signal Generator on the device:
0 - None
1 - Function Generator
2 - Arbitrary Waveform Generator
startFrequency Double (Bounded) 0.0 1000000.0 1000 Yes The frequency that the signal generator will initially produce (hertz).
stopFrequency Double (Bounded) 0.0 1000000.0 1000 Yes The frequency at which the sweep reverses direction or returns to the initial frequency (hertz).

Use the set and get functions, where appropriate, to change the value of a property or retrieve it's value.

Notes:

  • To output a signal with constant frequency, ensure that stopFrequency is equal to startFrequency.
  • The magnitude of the sum of offsetVoltage and peakToPeakVoltage should not exceed 4000 millivolts.

Additional Information:

The range of allowable frequencies for the startFrequency and stopFrequency are between one of these constants:

  • PS2000A_MIN_FREQUENCY
  • PS2000A_PRBS_MIN_FREQUENCY

and one of these constants:

  • PS2000A_SINE_MAX_FREQUENCY
  • PS2000A_SQUARE_MAX_FREQUENCY
  • PS2000A_TRIANGLE_MAX_FREQUENCY
  • PS2000A_SINC_MAX_FREQUENCY
  • PS2000A_RAMP_MAX_FREQUENCY
  • PS2000A_HALF_SINE_MAX_FREQUENCY
  • PS2000A_GAUSSIAN_MAX_FREQUENCY
  • PS2000A_PRBS_MAX_FREQUENCY

depending on the signal type. These values are defined in the PS2000aConstants class which is included in this toolbox.

Examples:

    % Obtain the arbitrary waveform buffer size for the device
    awgBufferSize = get(sigGenGroupObj, 'awgBufferSize');

    % Set the start frequency of the signal generator
    set(sigGenGroupObj, 'startFrequency', 5000.0);
